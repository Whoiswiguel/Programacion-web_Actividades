Carpeta completa generada para Programación Web.

Contenido:
- Basic_HTML: ejercicios base de HTML, listas, enlaces, estructura semántica y sitio simple.
- Basic_CSS: estilos inline, embebidos, externos, selectores, cascada y páginas con CSS.
- Basic_Layout: box model, float, clear, overflow, box-sizing, layouts de columnas y parallax.
- Basic_Tables: tablas, rowspan, estilos CSS, pseudo-clases y secciones thead/tbody/tfoot.
- Basic_Responsive_Layout: flexbox, grid, media queries e imágenes responsivas.
- Basic_JS: JavaScript básico, eventos, variables, funciones, validación y jQuery.

Puedes abrir el archivo index.html de esta carpeta para navegar por todas las prácticas.
Las imágenes faltantes fueron reemplazadas por imágenes generadas de apoyo con el mismo nombre que piden los ejercicios.
